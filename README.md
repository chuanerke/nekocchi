# nekocchi

<img src="show.gif"/>

A recreation of oneko for X11 in the Xlib library for C. 
Made to understand the X11 shape extension and Xlib even further. The assets were sourced from oneko-sakura, to which the rights belong. 
Compile with `gcc -Wall -Wextra -Wno-unused-variable -std=gnu99 -lX11 -lXext -lm -o neko neko.c`


[oneko-sakura](https://www.freshports.org/games/oneko-sakura/)
